# Block-Generator-Web

### [Link](https://han031121.github.io/Block-Generator-Web/)

### How to use

You can generate a block, adjust it to your preferred appearance, and save it as an image.

1. Set generation conditions

    Set the generation count, size, density, and other conditions, then select Start.

2. Browse blocks

    Use the arrow buttons to view the generated blocks one at a time.

3. Adjust the view

    In the Rendering tab, adjust the colors, camera, and lighting. Drag the block image to rotate it, and scroll to zoom in or out.

4. Save as JPG

    To save the block image, select the JPG button.
